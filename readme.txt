== Installation ==

If you know how WordPress works and also familier with WordPress Daseboard, then you will be able to install "And IT Portfolio Addon" without any issues. Just follow below step to install "And IT Portfolio for Elementor" Addon:

1. Go to your WordPress Dashboard and Search for "And IT Portfolio for Elementor"
2. Click on Install button and the addon will be installed within few seconds.
3. Once Installation completed, you will see activate button, click on it to activate the Plugin.
4. And you know what? Plugin is now ready to use. :)

= Old Way: =
1. Unzip (if it is zipped) and Upload `andit-portfolio-for-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= Configure =
1. After install and activate the plugin you will get a notice to install Elementor Plugin ( If allready install it then do not show notice. ).
2. To install the plugin click on the "Button" Install Elementor.
3. And IT Portfolio Widget list will be appear in Elementor page Editor
